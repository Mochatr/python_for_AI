# Mini Projet - Implémentation d'un Mini Réseau de Neurones avec Python et NumPy

## Objectif

Concevoir, implémenter et analyser un réseau de neurones multicouche à partir de zéro en utilisant uniquement NumPy et Matplotlib afin de comprendre les principes fondamentaux de l'apprentissage profond.

## Travail demandé
1. Implémenter les fonctions d'activation :
Sigmoid
Tanh
ReLU
Leaky ReLU

3. Visualiser les fonctions d'activation :

Tracé comparatif sur une même figure.
Grille de sous-figures 2×2.
3. Implémenter un réseau de neurones multicouche de type :  Entrée → Couche cachée → Sortie
4. Réaliser la propagation avant (Forward Propagation).
5. Implémenter l'algorithme de rétropropagation (Backpropagation).
6. Entraîner le réseau sur un petit jeu de données synthétique.
7. Etudier l'influence :

du taux d'apprentissage
du nombre de neurones cachés
du choix de la fonction d'activation
8. Evaluer les performances du modèle : Accuracy, Matrice de confusion, Courbe ROC.
